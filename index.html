Q-2: What is Event Bubbling in JavaScript? Explain with impressive example

Event bubbling is when an event triggered on a nested (child) element bubbles up to its parent elements. This allows handling events higher up in the DOM tree, which is useful in dynamic UIs.

ðŸ“Œ Real-World Example: Delete cards using bubbling 

<div id="cardContainer">
  <div class="card">
    <p>Card 1</p>
    <button class="deleteBtn">Delete</button>
  </div>
  <div class="card">
    <p>Card 2</p>
    <button class="deleteBtn">Delete</button>
  </div>
</div>

<script>
  document.getElementById(\"cardContainer\").addEventListener(\"click\", function(e) {
    if (e.target.classList.contains(\"deleteBtn\")) {
      e.target.parentElement.remove(); // remove entire card
      console.log(\"Card deleted via bubbling\");
    }
  });
</script>

  • Uses event delegation via bubbling
  • Efficient: only one listener for many delete buttons
  • Scalable: handles dynamically added cards too
  • Demonstrates real-world skill in UI logic

Q-4: What is Event Delegation in JavaScript? Give a real-world example

Event delegation is a technique in which a single event listener is added to a parent element to handle events on its child elements. It works using event bubbling and improves performance and flexibility.

ðŸ“Œ Example: Delete list items using delegation

<ul id="taskList">
  <li>Task 1 <button class="deleteBtn">Delete</button></li>
  <li>Task 2 <button class="deleteBtn">Delete</button></li>
  <li>Task 3 <button class="deleteBtn">Delete</button></li>
</ul>

<script>
  document.getElementById(\"taskList\").addEventListener(\"click\", function(e) {
    if (e.target.classList.contains(\"deleteBtn\")) {
      e.target.parentElement.remove(); // Delete the list item
      console.log(\"Task removed using event delegation\");
    }
  });
</script>

Q-5: How many types of JavaScript events are there? Explain with examples

JavaScript provides many types of events based on user interaction and browser activity. Here's a categorized list with real examples:

  • 1. Mouse Events:
    click, dblclick, mousedown, mouseup, mousemove, mouseover, mouseout 
    <button onclick="alert('Clicked!')">Click Me</button>
  • 2. Keyboard Events:
    keydown, keyup, keypress 
    <input type="text" onkeydown="console.log('Key pressed')" placeholder="Type here">
  • 3. Form Events:
    submit, change, focus, blur, input 
    <form onsubmit="event.preventDefault(); alert('Form submitted!')">
  <input type="text" name="name">
  <button type="submit">Submit</button>
</form>
  • 4. Window Events:
    load, resize, scroll, unload 
    <script>
  window.addEventListener("resize", () => {
    console.log("Window resized");
  });
</script>
  • 5. Clipboard Events:
    copy, cut, paste 
    <input onpaste="alert('Pasted!')" placeholder="Try pasting something">
  • 6. Media Events:
    play, pause, ended, volumechange 
    <video controls onplay="console.log('Video playing')" width="300">
  <source src="video.mp4" type="video/mp4">
</video>
  • 7. Drag and Drop Events:
    dragstart, drag, dragover, drop, dragend 
    <div draggable="true" ondragstart="console.log('Drag started')" style="width:100px;height:100px;background:#ccc;">
  Drag me
</div>
  • 8. Focus Events:
    focus, blur 
    <input onfocus="console.log('Focused!')" onblur="console.log('Blurred!')" placeholder="Focus me">
